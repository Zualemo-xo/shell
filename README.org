* Shell
A simple shell to execute commands.
** Limitations
- No pipes
- No redirection
- No quoting
- Does not interpret regular expressions
** Features
- Commands can be aliased
- Auto complete
- Command history
** Installation
#+BEGIN_SRC sh
# Compile
make

# Execute
make run
#+END_SRC
