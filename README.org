* Shell
A simple shell to execute commands.
#+CAPTION: Shell Demo
[[https://media.giphy.com/media/nbcE4lTCPJg3rEsoZG/giphy.gif]]
** Limitations
- No pipes
- No redirection
- No quoting
- No regular expressions
- No signal handling
** Features
- Command substitution 
- Auto complete
- Command history
** Installation
#+BEGIN_SRC sh
# Compile
make

# Execute
make run
#+END_SRC
