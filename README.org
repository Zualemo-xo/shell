* Shell
A simple shell to execute commands.
#+CAPTION: Shell Demo
[[https://media.giphy.com/media/nbcE4lTCPJg3rEsoZG/giphy.gif]]
** Limitations
- No pipes
- No redirection
- No quoting
- No signal handling
** Features
- Command substitution 
- Auto complete
- Command history
- String expansion like a posix shell
** Installation
#+BEGIN_SRC sh
# Compile
make

# Execute
make run
#+END_SRC
